Prediction Projects

  • Tech Stack: Python, Jupyter Notebook, Scikit-learn, Pandas, NumPy, Matplotlib, Random Forest, SVM, Logistic Regression, Gradient Boosting
  • GitHub URL: Project Link

This repository contains a collection of machine learning projects focused on supervised prediction tasks, each organized in separate folders with Jupyter Notebooks, datasets, and results. The projects address real-world use cases, leveraging classification and regression techniques.

Key projects include: Breast Cancer Prediction (binary classification using Wisconsin Breast Cancer Dataset), E-Commerce Product Delivery Prediction (predicting delivery times), Red Wine Quality (regression on UCI Red Wine Quality Dataset), Salary Prediction (regression with simulated data), Sleep Disorder Prediction (multi-class classification), and Titanic Survival Prediction (binary classification using Kaggle Titanic Dataset).
