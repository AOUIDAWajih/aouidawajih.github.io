MNIST Handwritten Digit Classification

  • Tech Stack: Python, Scikit-learn, TensorFlow, Keras, Jupyter Notebook, Matplotlib
  • Github URL: Projet sur GitHub

Ce projet est une implémentation de modèles de vision par ordinateur pour la classification d’images manuscrites issues du dataset MNIST, souvent considéré comme le “Hello World” du machine learning en vision.

Il s’adresse aux personnes ayant une première expérience avec Python ou R, qui souhaitent découvrir la vision par ordinateur et les techniques classiques comme les réseaux de neurones, SVM ou K-NN sur un jeu de données standard.

Le dataset MNIST contient des images 28x28 pixels de chiffres manuscrits de 0 à 9. Le but est de prédire correctement le chiffre représenté par chaque image.

Objectifs pédagogiques :

  • Maîtriser les bases de la vision par ordinateur
  • Implémenter des modèles de classification supervisée
  • Comparer différents algorithmes : régression, SVM, K-NN, réseaux de neurones
