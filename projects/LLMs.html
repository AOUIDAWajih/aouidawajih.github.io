Fine-Tuning Mistral-7B with PEFT and QLoRA

  • Tech Stack: Python, Transformers (Hugging Face), PEFT, QLoRA, TRL (SFTTrainer), bitsandbytes, accelerate, datasets
  • GitHub URL: Project Link

This project demonstrates fine-tuning the Mistral-7B-Instruct model using QLoRA (Quantized LoRA) and Parameter-Efficient Fine-Tuning (PEFT) on the iamtarun/python_code_instructions_18k_alpaca dataset. The implementation leverages the Hugging Face Transformers library and TRL's SFTTrainer for efficient training.

The workflow includes data preprocessing, model fine-tuning with quantization for efficiency, and support for multi-GPU training via the accelerate library. The project showcases advanced techniques for optimizing large language models for specific tasks, such as generating Python code instructions.
