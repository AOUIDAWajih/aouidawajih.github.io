Classification des chiffres manuscrits MNIST

  • Technologies utilisées: Python, Jupyter Notebook, Scikit-learn, TensorFlow/Keras, NumPy, Pandas, Matplotlib
  • URL GitHub: Lien du projet

Ce projet met en œuvre des modèles de vision par ordinateur pour classer les chiffres manuscrits à partir du jeu de données MNIST, une référence en apprentissage automatique pour la vision par ordinateur. Il constitue une introduction aux techniques telles que les réseaux neuronaux, les SVM et les K-plus proches voisins (K-NN) pour ceux ayant des connaissances de base en Python ou R et en apprentissage automatique.

Le jeu de données MNIST contient des images en niveaux de gris de 28x28 pixels représentant des chiffres manuscrits (0–9). L'objectif est de prédire le chiffre correct à partir d'une image. Le projet inclut un Jupyter Notebook avec le code, les visualisations et l'entraînement des modèles, comparant les performances de divers algorithmes comme la régression, les réseaux neuronaux, les SVM et les K-NN. Les principaux objectifs d'apprentissage incluent la maîtrise des bases de la vision par ordinateur et la mise en œuvre de modèles de classification supervisée.
