Fine-tuning BERT for Customer Review Sentiment Analysis

  • Tech Stack: Python, Jupyter Notebook, Transformers (Hugging Face), TensorFlow/PyTorch, Pandas, NumPy, Scikit-learn
  • GitHub URL: Project Link

This project demonstrates fine-tuning a BERT model for sentiment analysis of customer reviews using the Yelp dataset. The notebook, FineTuning_Transformer_BERT_Customer_Review.ipynb, provides a complete workflow including data loading and preprocessing, BERT model training, performance evaluation, and prediction on new reviews.

The Yelp dataset (yelp.csv) contains customer reviews with star ratings (1 to 5, mapped to 0 to 4 for model compatibility). The pipeline includes preprocessing, training, and evaluation using metrics like confusion matrices and classification reports. Key features include classifying sentiments based on star ratings and enabling predictions on new customer reviews.
