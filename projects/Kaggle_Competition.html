Automatic Summarization of Medical Scientific Articles with LLM

  • Tech Stack: Python, Jupyter Notebook, Transformers (Hugging Face), T5/BART/GPT, Pandas, NumPy, Scikit-learn
  • Kaggle Competition: Competition Link
  • GitHub URL: Project Link

This project develops an automatic summarization system using Large Language Models (LLMs) to synthesize medical scientific articles, specifically randomized controlled trials (RCTs) and observational studies. The system, optimized for the Kaggle M2 competition, achieved 1st place by producing reliable summaries evaluated using the ROUGE metric.

The workflow, implemented in a Jupyter Notebook, includes careful preprocessing of scientific texts, fine-tuning LLMs like T5, BART, or GPT, and evaluating performance with ROUGE-1, ROUGE-2, and ROUGE-L scores. The dataset, sourced from the Kaggle competition, includes full-text articles, expert-written reference summaries, and study metadata.
