Time Series Demand Forecasting with XGBoost

  • Tech Stack: Python, Jupyter Notebook, XGBoost, Pandas, NumPy, Matplotlib, Scikit-learn
  • GitHub URL: Project Link

This project demonstrates the use of the XGBoost algorithm for time series demand forecasting using the PJME_hourly dataset. The Jupyter Notebook, XGBoost_for_Time_Series_Dataset_DemandForecasting.ipynb, covers data preprocessing, model training, prediction, evaluation, and visualization of actual vs. predicted values and error analysis.

The PJME_hourly dataset (PJME_hourly.csv) contains hourly energy consumption data (in MW) for the PJME region, indexed by a datetime column, with PJME_MW as the target variable. The project provides a comprehensive workflow for building and evaluating a time series forecasting model.
